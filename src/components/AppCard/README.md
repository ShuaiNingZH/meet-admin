# AppCard 卡片组件

## 组件简介

AppCard 是一个基于 Element Plus
的卡片容器组件样式的二次封装，提供了更简洁的使用方式和更灵活的样式控制。该组件可用于包含各种内容的卡片式布局，适用于信息展示、数据统计等场景。

## 功能特点

- 简洁易用：提供简单的API，易于集成
- 阴影控制：支持三种阴影显示模式（始终显示、悬停显示、从不显示）
- 内边距自定义：支持自定义内边距大小

## 使用方法

```vue
<script setup lang="ts">
import { AppCard } from '@/components/AppCard';
</script>

<template>
  <!-- 基本用法 -->
  <AppCard>
    <div>基本卡片内容</div>
  </AppCard>

  <!-- 自定义阴影和内边距 -->
  <AppCard shadow="hover" :padding="30">
    <div>自定义卡片内容</div>
  </AppCard>
</template>
```

## 参数说明

### Props

| 参数名     | 类型               | 默认值      | 必填 | 说明                                                      |
|---------|------------------|----------|----|---------------------------------------------------------|
| shadow  | string           | 'always' | 否  | 卡片阴影显示时机，可选值：'always'(始终显示)、'hover'(悬停显示)、'never'(从不显示) |
| padding | string \| number | 20       | 否  | 卡片内边距，单位为像素                                             |

### Slots

| 插槽名     | 说明     |
|---------|--------|
| default | 卡片内容区域 |

## 注意事项

- 组件默认继承了 Element Plus 的 el-card 类，保留了基础样式
- 在处理大量内容时，建议设置适当的 padding 值，以提供更好的视觉效果

## 使用场景

AppCard组件适用于以下场景：

1. 信息展示卡片
2. 数据统计面板
3. 表单分组容器
4. 功能模块分区

## 内部实现

AppCard 组件内部基于 Vue 3 的 Composition API 实现，主要包含以下几个部分：

1. Props定义：控制卡片的阴影效果和内边距
2. 计算属性：动态计算卡片的样式类和内边距值
3. 模板结构：提供插槽用于内容放置
4. 样式定义：基于 SCSS 的样式封装
